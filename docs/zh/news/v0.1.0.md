# v0.1.0 发布说明

**发布日期：** 2026-03-26

Lumina v0.1.0 是首个公开版本，完成了从效果引擎到发布流程的全部基础建设。

## Phase 1 — 基础层

- x64 内联 Hook 引擎
- Win32 / DWM P/Invoke 定义
- OS 构建号检测
- `udwm.dll` 符号偏移表
- DLL 注入到 `dwm.exe`

## Phase 2 — 效果核心

- SystemBackdrop 效果（Mica / Acrylic / MicaAlt）
- AccentBlur 效果（Hook `CAccent::UpdateAccentPolicy`）
- CustomBlur 效果（Hook `CTopLevelWindow`）
- Aero 反射 + 视差效果
- 模糊半径 / 混合色自定义
- 亮色 / 暗色模式自动切换

## Phase 3 — GUI 与 SDK API

- 系统托盘图标 + 快速切换
- 设置窗口（纯 Win32，Native AOT 兼容）
- 效果预设切换
- 混合色拾色器
- `Lumina.Core` 标准模式 API
- `Lumina.Core.Advanced` 高级模式 API
- `Lumina.WinForms` 扩展方法适配器
- `Lumina.Wpf` 附加属性适配器

## Phase 4 — 配置与文档

- JSON 配置文件读写
- 多语言支持（中文 / 英文）
- 开机自启
- 导入 / 导出配置
- `EffectProfile` JSON 序列化
- XML 文档注释（IntelliSense）
- 排除窗口列表 API

## Phase 5 — 发布

- Inno Setup 安装包（`installer/Lumina.iss`）
- MSIX 打包配置（`msix/Package.appxmanifest`）
- 崩溃转储收集（`CrashHandler.cs`）
- 节能模式（`PowerMonitor.cs`）
- 单元测试项目（`Lumina.Tests`）
- GitHub Actions CI：构建 + 测试 + NuGet 发布 + 安装包 + GitHub Release
- 文档站（JekyllNet，中英双语，`lumina.maikebing.com`）

## NuGet 包

| 包名 | 说明 |
| --- | --- |
| `Lumina` | 标准模式核心 API |
| `Lumina.Advanced` | 高级模式（需管理员权限） |
| `Lumina.WinForms` | WinForms 扩展方法 |
| `Lumina.Wpf` | WPF 附加属性 |

## 已知限制

- 仅支持 x64 架构
- Aero 玻璃效果仅在高级模式下可用
- MSIX 图标素材尚待设计（占位符）
- 第三方主题兼容性测试用例待补充
